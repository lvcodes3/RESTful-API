- Creating a RESTful API using Node.js & Express.js

    1. Inside of the main directory in terminal: "npm init -y"
       This initializes a new Node.js program.

    2. "npm install express cors lodash uuid"

    3. We then set up the start script in our package.json by adding: "start": "node index.js",
       to the scripts object

    4. We can then "npm start" in order to successfully run our newly created index.js
